A brief history of JavaScript

  • Prototype based, Object oriented scripting language
  • Released in 1995 (with many bugs and no time to fix!)
  • Progress: Basic validations->Interactions->RIA->Libraries...
  • Industry standard - ECMA 262 specification
  • Several dialects-JavaScript, JScript, ECMAScript..
  • ECMAScript 3rd Edition (ES3) - December 1999
  • Latest standard - ECMAScript 5th Edition (ES5) - December 2009
  • ES5 fixes bad parts of JavaScript and adds new APIs.
